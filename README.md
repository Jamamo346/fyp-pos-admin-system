# Final Year Project - POS Administration System

## project title 
Evaluating Privacy Protections in AI-Powered Cloud Services within the Retail Industry: A Network Forensics Case Study of POS Administration Systems
## Features
- Role-based access (Admin, Manager, Cashier)
- POS transaction system
- Dashboard with KPIs
- Inventory management
- AI Data Assistant (Python backend)

 ## Tools
- PHP
- MySQL
- Python
- HTML/CSS

## Security Testing
Wireshark was used to analyse HTTP vs HTTPS traffic.
